# 前端使用说明

## 环境要求

- Node.js 16+ 
- npm 或 yarn

## 安装依赖

```bash
cd frontend
npm install
```

## 开发运行

```bash
npm run dev
```

访问地址: http://localhost:5173

## 构建生产版本

```bash
npm run build
```

## 功能说明

### 买家端功能
1. **商品列表** - 浏览所有商品，支持关键词、类目、价格搜索
2. **为你推荐** - 基于协同过滤算法的个性化推荐
3. **我的订单** - 查看订单历史

### 商家端功能
1. **数据看板** 
   - 销售趋势折线图（支持7/30/90天）
   - 热销商品Top10柱状图
   - 类目销售分布饼图
   - AI经营建议

2. **商品管理** - 新增、编辑、删除商品

3. **订单管理** - 查看所有订单

## 注意事项

1. 确保后端服务运行在 http://localhost:8000
2. 首次运行需要安装依赖 `npm install`
3. 如果端口被占用，可在 vite.config.js 中修改端口
